HTML Elements

Element Description
Heading Elements Defines headings (e.g., <h1> to <h6>)
Paragraph Element Defines a paragraph of text (<p>)
Anchor Element Defines a hyperlink (<a>)
Image Element Embeds an image (<img>)

HTML Tags

Tag Description
<div> Defines a division or section in a document
<span> Used for grouping inline-elements for styling
<ul> Defines an unordered (bulleted) list
<table> Defines a table

HTML Attributes

Attribute Description Example
href Specifies the URL for an anchor tag (<a>) <a href="https://example.com">Link</a>
src Specifies the source URL for an image tag (<img>) <img src="image.jpg" alt="Description">
alt Provides alternate text for an image if it cannot be displayed <img src="image.jpg" alt="Description">
class Specifies a class name for CSS styling <div class="container">Content</div>
